## Supported Formats

NAIIVE is built to handle large-scale data efficiently. 
It currently supports the following formats commonly used in my research workflow:

* **LAMMPS:** `.dump` (text dump with `.gz` support)
* **DeepMD:** `deepmd/npy`, `deepmd/raw` (Machine Learning Force Field Datasets)
* **GAMESS:** `.inp` (Input files)
* **VASP:** `POSCAR`, `CONTCAR`, `XDATCAR`
* **VASP:** `CHGCAR` (Volumetric data)
