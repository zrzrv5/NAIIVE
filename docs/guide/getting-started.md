# Getting Started

Welcome to the Naiive documentation.

## What is Naiive?

Naiive is an experimental node-based visualization studio built with Swift & Metal. It is designed to explore real-time molecular rendering across Apple platforms.

## Installation

Naiive is currently in active early development.

You can try the latest build on [TestFlight](https://testflight.apple.com/join/VrYnN9nE).

The application is free to use and will be available on the App Store upon official release.
